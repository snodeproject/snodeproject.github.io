Perché Snode?

Snode offre un ambiente familiare per chi conosce Node.js, ma è interamente sviluppato in Python come progetto open source.

Runtime JavaScript

Esegui codice JavaScript tramite un runtime sviluppato in Python.

SPM

Installa e gestisci moduli con lo Snode Package Manager.

require()

Importa moduli utilizzando un sistema ispirato a Node.js.

Documentazione

Consulta guide, API ed esempi per iniziare rapidamente.

Open Source

Snode è distribuito sotto licenza MIT e sviluppato pubblicamente.
